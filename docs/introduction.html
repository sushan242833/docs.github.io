Introduction

This site contains the documentation for my BOOSE interpreter application.

What this application does

The BOOSE application is a small interpreter that parses and executes BOOSE language programs to draw on a Windows Forms canvas. It supports BOOSE facilities such as:

  • Variables: int, real, boolean
  • Arrays: array int, array real
  • Array access: poke, peek
  • Control flow: if, else, end if, while, for
  • Methods: method, call
  • Drawing commands (canvas output): e.g. circle, rect, moveto, drawto, pen, write

Project structure

  • BooseApp/: main Windows Forms application + command implementations
  • TestBooseApp/: MSTest project containing facility tests (unit/integration style)

Where to go next
